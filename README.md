- Project description
KeenKeeper is a dedicated platform for nurturing personal relationships. Built using Next.js and Tailwind CSS, it enables users to log interaction timelines, track audio/video calls, and review connection statistics through beautifully designed interactive charts. It functions as a personal dashboard to ensure you stay in touch and maintain balance in your social life.

##- Technologies used
Frontend: Next.js (App Router), React, Tailwind CSS, Daisyui 
Data Visualization: Recharts (Dynamic Pie Charts & Analytics)
 
🌟 Key Features of my project
***Interactive Interaction Timeline: Keeps a structured, chronologically sorted log of all interactions, including texts, audio calls, and video calls with specific timestamps.
***Dynamic Data Visualization: Utilizes Recharts to render full-screen responsive Pie Charts that offer real-time analytics and break down your communication patterns visually.
***Granular Communication Filtering: Easily categorizes and separates interaction data based on the action type (Call, Text, Video Call) for quick filtering.
***Modern Fluid UI/UX: Designed with a clean, minimalist aesthetics using Tailwind CSS, featuring smooth hover animations 
***Fully Responsive & Accessible: Completely optimized for all screen sizes, adapting fluidly from narrow mobile viewports to expansive high-resolution desktop displays.
